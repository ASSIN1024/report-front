## 1. 设计原则

### 1.1 总体原则

| 原则 | 说明 |
|------|------|
| **一致性** | 整个应用保持统一的视觉和交互风格 |
| **简洁性** | 界面简洁明了，避免过度设计 |
| **可用性** | 操作符合用户预期，减少学习成本 |
| **响应性** | 适配不同屏幕尺寸 |

### 1.2 设计参考

- Element UI 官方设计规范
- 简洁实用的企业级后台管理系统风格

### 4.1 整体布局

```

│                    Header (可选)                         │
├──────────────┬────────────────────────────────────────┤
│              │                                          │
│   侧边栏     │              主内容区域                    │
│  (200px)    │                                          │
│              │                                          │
│              │                                          │
```

每个页面包含以下区域：

| 区域 | 说明 |

## 5. 组件规范

### 5.1 按钮

| 类型 | 样式 | 使用场景 |
|------|------|----------|
| 主按钮 | 蓝色实心 | 主要操作 |
| 默认按钮 | 白色边框 | 次要操作 |
| 文字按钮 | 无边框 | 表格内操作 |

**按钮尺寸**:
- 中等 (默认): height 32px
- 小型: height 24px

### 5.2 表格

| 规范 | 说明 |
|------|------|
| 边框 | 带边框表格 |
| 斑马纹 | 使用斑马纹提高可读性 |
| 固定列 | 操作列固定在右侧 |
| 加载状态 | 使用 v-loading |

### 5.3 表单

| 规范 | 说明 |
|------|------|
| 标签宽度 | 120px |
| 输入框宽度 | 200px / 300px |
| 验证提示 | 表单项下方红色文字 |

### 5.4 弹窗

| 规范 | 说明 |
|------|------|
| 宽度 | 600px (标准) |
| 标题 | 顶部居中 |
| 关闭按钮 | 右上角 |
| 底部按钮 | 右下角对齐 |

### 5.5 分页

| 规范 | 说明 |
|------|------|
| 位置 | 右下角 |
| 背景色 | 白色背景 |
| 组件 | el-pagination (带background) |

## 7. 页面规范

### 7.1 FTP配置页面

**路径**: `/ftp`

**功能**:
- 配置列表展示
- 新增/编辑配置
- 连接测试
- 删除配置

### 7.2 报表配置页面

**路径**: `/report`

**功能**:
- 报表列表展示
- 新增/编辑报表配置
- 查看配置详情
- 删除配置

### 7.3 任务监控页面

**路径**: `/task`

**功能**:
- 任务列表展示
- 查看执行日志
- 重试失败任务
- 取消待执行任务
- 删除任务记录

### 7.4 执行日志页面

**路径**: `/log`

**功能**:
- 日志列表展示
- 按任务筛选
- 日志级别高亮

### 7.5 操作日志页面

**路径**: `/operation-log`

**功能**:
- 操作日志列表展示
- 按模块、操作类型、结果筛选
- 查看操作详情（操作前后数据对比）
- 支持分页查询

### 7.6 系统日志页面

**路径**: `/system-log`

**功能**:
- 日志文件列表展示
- 按日志类型筛选（全部/错误/操作/访问）
- 按级别、关键词搜索
- 日志内容实时查看
- 日志文件导出

## 8. 响应式设计

### 8.1 断点

| 断点 | 宽度 | 适配设备 |
|------|------|----------|
| xs | < 768px | 手机 |
| sm | 768px - 992px | 平板 |
| md | 992px - 1200px | 小屏电脑 |
| lg | > 1200px | 大屏电脑 |

### 8.2 侧边栏

- 固定宽度 200px
- 最小高度 100vh
- 可折叠 (当前版本固定显示)
