## 3. 评估方法

### 3.1 评分标准

| 等级 | 分数 | 说明 |
|------|------|------|
| 优秀 | 5 | 准确无误，并能举一反三 |
| 良好 | 4 | 准确无误 |
| 合格 | 3 | 基本准确，有轻微遗漏 |
| 不合格 | 1-2 | 错误或严重遗漏 |
| 失败 | 0 | 完全错误或无法回答 |

### 3.2 评估流程

1. 读取测试问题
2. 思考预期回答
3. 评估实际回答
4. 给出评分和评语
5. 记录到评估报告
